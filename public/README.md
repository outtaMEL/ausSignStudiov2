# Public Assets

这个目录包含静态资源文件。

## Favicon

项目包含以下favicon文件：

- `favicon.ico` - 传统ICO格式（兼容所有浏览器）
- `favicon.svg` - 现代SVG格式（支持暗黑模式和高分辨率）

### 自定义Favicon

如果你想替换favicon：

1. 准备你的图标文件（推荐32x32px或更大）
2. 替换 `public/favicon.ico` 和 `public/favicon.svg`
3. 重启开发服务器

### 生成Favicon工具

推荐使用以下工具生成多种格式的favicon：
- https://realfavicongenerator.net/
- https://favicon.io/

## 其他静态资源

你可以在这个目录添加其他静态文件：
- 图片
- 字体
- robots.txt
- sitemap.xml
- 等等

这些文件可以通过 `/文件名` 直接访问。
